# Autonomous Regenerative IoT Vacuum Vehicle

## Description
This project develops an autonomous vacuum-cleaning vehicle with IoT-based monitoring and regenerative energy harvesting to enable efficient, self-operating cleaning with real-time performance tracking and improved energy utilization.

## Features
- Autonomous navigation with obstacle detection
- Vacuum-based cleaning system
- Regenerative energy generation
- IoT monitoring using ESP8266

## Technologies Used
- Arduino Uno
- NodeMCU ESP8266
- Embedded C
- IoT Cloud Platform

## How It Works
The robot detects obstacles using ultrasonic sensors, performs cleaning using a vacuum mechanism, generates energy through wheel motion, and sends real-time data to the cloud.

## Future Improvements
- Mobile app integration  
- Battery charging using regenerated energy  
- Advanced navigation (AI/SLAM)
